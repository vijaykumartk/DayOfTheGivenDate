# Day Of The Given Date

This Python application calculates the day of the week for any given date. 

## Features
- Input any date in the format DD-MM-YYYY.
- Outputs the corresponding day of the week.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/DayOfTheGivenDate.git
    ```
2. Navigate to the project directory:
    ```bash
    cd DayOfTheGivenDate
    ```
3. Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
Run the application using:
```bash
python app.py
```
Follow the prompts to enter a date.
